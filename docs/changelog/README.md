## 变更记录规范

新的变更条目请遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/) 格式：

- **Added** — 新增功能
- **Changed** — 已有功能的变更
- **Deprecated** — 即将移除的功能
- **Removed** — 已移除的功能
- **Fixed** — Bug 修复
- **Security** — 安全相关修复

重大修复或排障过程建议单独创建日期命名的文档（如 `2026-03-29-startup-fix.md`）。
