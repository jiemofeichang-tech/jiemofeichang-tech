### 1. script_engine.py
Python 模块，实现剧本生成：
- class ScriptEngine(api_key, base_url, model='claude-opus-4-5')
- generate_script(params: dict) -> dict
  - params: {project_id, genre, theme, characters_count, episodes_count}
  - 调用 LLM，system prompt 要求严格输出 JSON

  - 解析 JSON（支持 markdown 代码块提取）
  - 保存到 D:/AiProject/AiComedyDrama/data/scripts/{script_id}.json
  - 返回完整 dict

- _parse_json_safe(text) -> dict: 提取 JSON，支持 ```json 代码块包裹

### 2. character_factory.py
Python 模块：
- class CharacterFactory(api_key, base_url, image_api_url)
- create_character(params: dict) -> dict
  - params: {project_id, name, personality, appearance_desc, role_type}
  - 生成 character_id
  - 构造 9 个图片生成 prompt（3视图+6表情）
  - 调用 image_api_url 提交图片任务
  - 保存初始状态到 D:/AiProject/AiComedyDrama/data/characters/{id}.json
  - 返回 {character_id, name, status: 'generating', tasks: {front, side, back, happy, sad, angry, surprised, thinking, shy}}
- get_character(character_id) -> dict: 读取JSON，轮询每个task状态，更新
- _submit_image_task(prompt) -> str: POST 到 image_api_url，返回 task_id
- _check_task_status(task_id) -> dict: 查询任务状态

图片 prompt 模板：
  front: {appearance}，正面全身角色设计图，动漫风格，白色背景，高清线稿，细节丰富
  side:  {appearance}，侧面全身角色设计图，动漫风格，白色背景，高清线稿
  back:  {appearance}，背面全身角色设计图，动漫风格，白色背景，高清线稿
  happy: {appearance}，脸部特写，开心灿烂笑容，动漫风格，白色背景
  sad:   {appearance}，脸部特写，悲伤流泪表情，动漫风格，白色背景
  angry: {appearance}，脸部特写，愤怒皱眉表情，动漫风格，白色背景
  surprised: {appearance}，脸部特写，惊讶张嘴表情，动漫风格，白色背景
  thinking:  {appearance}，脸部特写，思考歪头表情，动漫风格，白色背景
  shy:       {appearance}，脸部特写，害羞脸红微笑，动漫风格，白色背景

### 3. storyboard_generator.py
Python 模块：
- class StoryboardGenerator(api_key, base_url, image_api_url)
- generate_storyboard(params: dict) -> dict
  - params: {project_id, script_id, episode_num, style}
  - 读取 data/scripts/{script_id}.json
  - 对每个 scene 调用 _scene_to_panels() 生成 2-3 个分镜描述
  - 对每个分镜提交图片生成任务
  - 保存到 data/storyboards/{storyboard_id}.json
  - 返回 {storyboard_id, episode_num, panels:[{panel_id,scene_id,shot_type,image_prompt,task_id,status,image_url}]}
- _scene_to_panels(scene, style) -> list: 调用 LLM 生成分镜JSON数组
  每个分镜：{shot_type(特写/近景/中景/全景/远景), camera_angle(平视/仰视/俯视), image_prompt(英文), dialogue_ref, duration_sec}
- get_storyboard(storyboard_id) -> dict: 轮询更新各面板状态

### 4. video_composer.py
Python 模块：
- class VideoComposer(api_key, base_url, seedance_api_url)
- compose_video(params: dict) -> dict
  - params: {project_id, storyboard_id, voice_config: {speaker, speed}}
  - 读取分镜数据
  - 对每个分镜图片提交 Seedance 图生视频任务
  - 记录所有 task_id 到 data/video_tasks/{video_task_id}.json
  - 返回 {video_task_id, status: 'processing', panels_count}
- get_video_task(video_task_id) -> dict: 更新进度，全部完成后调用 _merge_with_ffmpeg
- _merge_with_ffmpeg(video_paths, output_path): 用 subprocess 调用 ffmpeg 合并
